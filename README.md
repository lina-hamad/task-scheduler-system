# Task Scheduler System (C)

This project implements a task scheduling system using data structures and file handling in C.

## Features
- Load tasks from file
- Manage and schedule tasks
- Process task data efficiently
- File-based input handling

## Files
- main.c → program source code
- tasks.txt → input data
- cities.txt → supporting data (if used)

## How to Run

Compile:
gcc main.c -o scheduler

Run:
./scheduler
